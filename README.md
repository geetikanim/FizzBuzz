# FizzBuzz

This is a program that prints numbers from 1 to 100 based on the following requirements:

Stage 1
- For multiples of three print "Fizz"
- For multiples of five print "Buzz"
- For multiples of both three and five print "FizzBuzz"

Stage 2
- A number is Fizz if it is divisible/multiple of 3 or if it has a 3 in it
- A number is Buzz if it is divisible/multiple of 5 or if it has a 5 in it
- A number is FizzBuzz if it satisfy above 2 criterias together

### Installation

Clone
- Clone this repo to your local machine using https://github.com/geetikanim/FizzBuzz.git

### Tech

This program uses following tools and languages: 
- Java version "1.8"
- Apache Maven 3.3.9
- junit 4.11

#### Building for source
```sh
$ mvn clean install
```

### Run as a Java Program

You can also run FizzBuzz main method as a java program using **mvn exec:java** command for the desired results 


### Tests

There are test cases written for each of the classes which can also be run for unit testing
